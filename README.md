# 🏨 Hotel Reservation System

## 📌 Description

This is a GUI-based Hotel Reservation System developed using Java Swing and JDBC.
The application allows users to manage hotel bookings and store data in a MySQL database.

---

## 🛠️ Technologies Used

* Java (Swing)
* JDBC (Java Database Connectivity)
* MySQL

---

## ⚙️ Features

* Add Reservation
* View Reservations
* Update Reservation
* Delete Reservation

---

## 🗄️ Database Structure

Table: `reservations`

* id (Primary Key)
* guest_name
* room_number
* contact
* room_type
* check_in_date

---

## ▶️ How to Run

1. Compile the program:

```
javac -cp ".;mysql-connector-j-8.0.33.jar" HotelMenuApp.java
```

2. Run the program:

```
java -cp ".;mysql-connector-j-8.0.33.jar" HotelMenuApp
```

---

## 🧠 Concept

This project demonstrates CRUD operations (Create, Read, Update, Delete) using Java GUI and MySQL database connectivity.

---

## 🎯 Output

The system provides a simple graphical interface to manage hotel reservations efficiently.
